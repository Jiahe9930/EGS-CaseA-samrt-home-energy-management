# EGS Coursework - Case A Smart Home Energy Management

This repository contains the code for my EGS individual coursework on Case A: Smart Home Energy Management with PV and battery storage.

## Contents
- `caseA_model.py` : main simulation and optimisation code
- `README.md` : project description
- `requirements.txt` : Python package requirements (if applicable)

## Project summary
This project compares:
1. A self-consumption-first dispatch policy
2. A tariff-aware optimisation policy
3. An extension with battery degradation cost

## Main outputs
- Energy breakdown
- Cost breakdown
- Battery SOC time series
- Verification checks
- Comparison between dispatch policies

## Notes
The code was developed for coursework submission and is shared for assessment purposes.
